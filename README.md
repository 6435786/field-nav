# field-nav

כלי ניווט שטח לעבודה מבצעית — המרת קואורדינטות, נקודות שמורות בפרוייקטים, אזימוט בלחיצה, וסריקת QR.

נבנה ראשית עבור פעילות יחידת רחפנים ומשימות חילוץ.

**URL:** https://6435786.github.io/field-nav/

## יכולות עיקריות

- זיהוי אוטומטי של WGS84 / ITM / UTM 36N / DMS לפי טווחי ערכים
- המרה הדדית בין כל הפורמטים
- גובה אמיתי מ-Open-Meteo (Copernicus DEM)
- 4 שכבות מפה: רחובות, לוויין, טופוגרפי, שבילי ישראל
- מעקב GPS חי עם משולש מצפן
- סריקת QR ממיקומים משותפים → הופך לנקודה
- שמירת נקודות עם שם בפרוייקטים
- אזימוט+מרחק בלחיצה: נקודה אחת מהמיקום שלך, שתיים בין הנקודות
- "פתח באפליקציית מפה" — `geo:` URI לבחירת אפליקציה ב-Android, share sheet ב-iOS

## טכנולוגיה

קובץ HTML יחיד, vanilla JS, ללא build. רץ ישירות מ-GitHub Pages.

ספריות (כולן מ-CDN):
- Leaflet 1.9.4
- proj4js 2.9.0
- qrcodejs 1.0.0
- html5-qrcode 2.3.8
- Firebase 9.23 (compat)
