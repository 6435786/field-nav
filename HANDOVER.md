**מותג + צבעים:** מותג=**טורקיז** (`--primary:#2a7068`), אפליקציה-רחבה. **כחול שמור ל"אתה"** (`--location:#1a56db`) — חץ המיקום + עיגול דיוק.

**סמני נקודות (נחתו):** עיגול כחול **מלא** + מסגרת לבנה. שמורה=קטן; נוכחית/חיפוש=גדול יותר + נקודה לבנה במרכז. מיקום-ידני=חץ כחול חלול. מדידה A/B=ירוק/אדום.

**כותרת מינימלית:** שם+לוגו הוסרו → **☰ (ימין) · 🔍 (מרכז) · אווטר-עיגול (שמאל)**. ה-☰ (`moreMenu`) מחזיק: בורר פרוייקט · ➕ חדש · ⚙️ הגדרות · 🛡️ ניהול (role-gated) · ייבוא/ייצוא/התקנה/לילה. אווטר→תפריט (פרופיל/יציאה).

**שורת קלט:** רק **[שדה קואורדינטות גדול] + [הוסף טורקיז]**. נקה ירד לסרגל; 🔍 עלה לכותרת.

**סרגל כלים:** 5 כפתורים בלי-מסגרת, אייקון-מעל-טקסט, אפור (נקה/מדידה/QR/ידני/מיקום) + קווים קצרים עדינים ביניהם.

**פקדי מפה:** "זכוכית מטה" `rgba(255,255,255,.38)`+`blur`. כפתורי-פעולה=עגולים; **שכבות+זום=מלבן** (50% ריסק את פאנל-השכבות הפתוח!). אייקונים טורקיז (כולל אייקון השכבות; 🎯→כוונת-SVG; הקלטה=אדום קטן). זום+שכבות נדחפו טיפה למטה.

**שכבות מפה:** הוסרה "טופוגרפי". "שבילי ישראל"→**"מפת שבילים"**. ברירת מחדל=מפת שבילים. נשארו 3: רחובות(OSM) / לוויין(Esri) / מפת שבילים(IHM).

**פתיחה ראשונית:** מתמרכז על מיקום המשתמש ב**זום 16** (`centerOnInitialLocation`, high-accuracy, מוגן ב-`_userMovedEarly`).

**אינטראקציה:** לחיצה על נקודה שמורה = פתיחת פרטים (הסרנו בחירת-1-2 לאזימוט; מדידה בין נקודות = 📏). **נעילת-מחווה במגע** (ב-`leaflet-rotate.js`): צביטה=זום (אפס סיבוב מקרי), פיתול=סיבוב (זום מותר תוך כדי); פיתול נמדד מהזווית האמיתית (תוקן באג +180), סף ~12-14°.

### ⏳ איפה עצרנו בדיוק (להמשיך מכאן)
המשתמש מרגיש שהמעבר בין הפאנל העליון למפה **"חתוך מדי"**. פסלנו "קצה נמוג" (gradient דוהה ללבן — מכוער על מפה צבעונית). **הסכמנו על "זכוכית": העליון (כותרת + 2 שורות) ירחף מעל המפה ב~65% + blur.** הצעתי לבנות — **ממתינים לאישור "כן".**
**להמשך בניית הזכוכית:**
- להציף את העליון מעל המפה (map-area ממלא מאחור) — **לתחום ל-mobile** (`@media max-width:899px`) שלא לשבור את גריד הדסקטופ.
- רקע: כותרת `rgba(42,112,104,~.65)`+blur · פאנל `rgba(255,255,255,~.65)`+blur (+override מצב-לילה). **שדה הקואורדינטות טיפה יותר אטום** לקריאוּת.
- **קריטי:** לדחוף את הזום+שכבות (`.leaflet-top`) למטה בגובה הפאנל (למדוד ל-CSS var), אחרת ייחבאו מתחת לפאנל הצף.

**פתוח נוסף:** תצ"א טוב יותר לישראל (govmap/Google/ספק-מפתח — הצעתי לחקור, לא נעשה) · באג ה-role-drop (ראה למעלה) · הקלטות-בענן / onboarding / iOS · רשימת ליטוש UI ישנה.

**שיטת עבודה:** edit → `node` syntax-check → commit → push → המשתמש מרענן+מצלם → מכווננים. כל שינוי = bump SW. קאש לפעמים מתעדכן בסבב-רענון שני.

## מה הושלם 2026-06-16 — B: סיבוב מפה + 3 מצבים + מעקב ✅

נדחף לפרודקשן ואושר ע"י המשתמש ("נראה פצצה"). קומיטים: `d72afe0` (מנוע) · `549972b` (כפתור+מצבים) · `7d7cd4a` (תיקון כיוון) · `e4d04a7` (רעידות) · `83f9b68` (סמן) · `c7d53b0` (מעקב). SW → v80.

**ספרייה:** `leaflet-rotate` ירדה **מקומית** ל-`C:/NAV/leaflet-rotate.js` (לא CDN — בגלל סינון רשת + אופליין), נוספה ל-precache של ה-SW. המפה מאותחלת עם `rotate:true, touchRotate:true, shiftKeyRotate:true, rotateControl:false` (אנחנו מנהלים את ה-bearing בעצמנו).

**3 מצבים** (`rotationMode`: 'north'|'manual'|'compass'):
- 🧭 כפתור (bottomleft ליד 🎯): צפון→מצפן · מצפן→צפון · ידני→צפון. מחט כחולה שמצביעה לצפון.
- סיבוב 2 אצבעות → ידני (מזוהה דרך אירוע `rotate` + דגל `_programmaticRotate` שמבדיל סיבוב שלנו מסיבוב משתמש).
- מצפן: `setBearing(360 − heading)` (heading-up), מונע ע"י DeviceOrientation.

**מעקב (🎯):** מדליק auto-center. כניסה: ידני→מצפן, צפון/מצפן נשארים. כיבוי: לחיצה שנייה על 🎯, או גרירת המפה (`dragstart`). ה-GPS/הסימון **לא** נכבים — רק המעקב מתחלף. הכפתור "פעיל" (כחול) = מעקב דלוק.

**מוסכמות סימן (קריטי — מתועדות בקוד):** leaflet-rotate מסובב את המפה עם כיוון השעון ב-`rotate(+bearing)`, וסמנים נשארים זקופים (`markerPane` תחת `norotatePane`). לכן: מחט=`rotate(+bearing)` · חץ-משתמש=`rotate(heading + bearing)` · מצפן=`setBearing(360 − heading)`. heading: iOS=`webkitCompassHeading`, אנדרואיד=`360 − alpha`.

**רעידות (תוקנו):** (1) מאזינים לחיישן כיוון **אחד** בלבד — absolute אם קיים אחרת relative (שניהם יחד התנגשו); (2) low-pass + rAF + dead-band על ה-heading (`scheduleHeadingUpdate` / `headingSmoothed`); (3) מצפן עדיף על GPS course (`orientationProviding`).

**סמן מיקום:** משולש כחול אחד (`.user-arrow-svg`, SVG path) במקום עיגול+חרוט. גרסת מיקום-ידני כתומה.

**+ באותו יום (המשך):**
- **הסרת שורת WGS84** מתחת לקלט המהיר — היתה הד מיותר של הקואורדינטה. השורה (`#detectedMini`) עכשיו `:empty{display:none}` ונשמרת רק להודעות מדידה/ידני/הרחבת-קישור. קומיט `0c4ac61`, v81.
- **חיפוש מקום (🔍 — #6 הושלם):** כפתור צף בפינה ימנית-עליונה של המפה → שורת חיפוש בשכבת-על → `GET israelhiking.osm.org.il/api/Search/{term}?language=he` (אותו שרת כמו אריחי "שבילי ישראל"; CORS עובד ישירות, אומת בשטח). תוצאה: `{displayName, location:{lat,lng}}` → `setActiveUnsavedPoint(lat,lng,true,name)` + `disableFollow()`. ב-SW: חוק האריחים צומצם ל-`israelhiking…/Hebrew` כדי שה-API יישאר network-only. קומיט `9d9e23a`, v82.
